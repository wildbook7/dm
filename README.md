WSL での始め方

- [Volta](https://volta.sh/) をインストール
- Volta を使って Node をインストール
- Volta を使って [pnpm](https://pnpm.io/ja/installation) をインストール
- [Claude Code](https://code.claude.com/docs/ja/overview) をインストール
- react-ts テンプレート指定にて [Vite プロジェクトを生成する](https://ja.vite.dev/guide/#%E6%9C%80%E5%88%9D%E3%81%AE-vite-%E3%83%95%E3%82%9A%E3%83%AD%E3%82%B7%E3%82%99%E3%82%A7%E3%82%AF%E3%83%88%E3%82%92%E7%94%9F%E6%88%90%E3%81%99%E3%82%8B)
- [Tailwind CSS](https://tailwindcss.com/docs/installation/using-vite) の設定
